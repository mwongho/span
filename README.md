
## About

A command-line application that will calculate the ranking table for a
soccer league.

### Notes:
* Prerequisites: Java 8+ && maven 3+
* Build: `$>mvn package`
* Run: execute from project root directory where pom.xml is located
    * arg0 : input file
    * arg1 : output file
    * `$>mvn exec:java -Dexec.args="arg0 arg1"`
